Como usar: dois modos

O mesmo pacote funciona de dois jeitos. Escolha o seu.

No claude.ai

Para qualquer pessoa, sem instalar nada

  1. Crie uma conta no claude.aiPara publicar a página no final, use um plano pago, o mais básico resolve.
  2. Ative a busca na web na conversaÉ ela que faz a pesquisa de mercado de verdade.
  3. Cole a mensagem de ativação abaixo e envieO Claude lê o pacote e vira o seu assistente na hora.

No Claude Code

Para quem já usa o terminal · modo squad

  1. Baixe ou clone este pacoteTodos os arquivos desta pasta, do jeito que estão.
  2. Abra o terminal na pasta e rode claudeO assistente assume sozinho. Diga oi e comece.
  3. Trabalhe com o squadUm condutor conversa com você e 8 agentes especialistas produzem as entregas.

Vantagens do modo squad: cada entrega vira arquivo na pasta producao/ (caiu a conversa, retoma do ponto exato) e a página pode ir pro ar com URL própria via npx vercel producao/.

Três coisas para saber

a
Se travar, destrava fácil

Recarregue a página e cole na conversa: "estamos retomando, continua de onde paramos". O assistente lembra tudo. No modo squad, é só abrir a pasta de novo: a jornada está salva em producao/.

b
Responda com sinceridade, não com perfeição

Não trave buscando a resposta ideal. O papel do assistente é organizar o que você sabe. A página que sai hoje vale mais que a ideia perfeita que nunca sai do papel.

c
A primeira versão é o ponto de partida

Tudo o que sair pode ser ajustado depois: copy, cores, preço, anúncios. Nada se perde, tudo se melhora.
